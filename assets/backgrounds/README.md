# 背景画像 / Background Images

このフォルダには、ゲーム内の各画面で使用される背景画像が格納されています。

## 画像仕様 / Image Specifications

- **サイズ**: 1280 x 720 ピクセル (PNG形式)
- **形式**: PNG (透明背景非対応 - 完全な背景画像として使用)
- **アスペクト比**: 16:9

## 背景画像リスト / Background Image List

現在のプレースホルダー画像は以下の画面用に作成されています：

| ファイル名 | 用途 | 画面 | 色調 |
|-----------|------|------|------|
| main_background.png | メイン画面の背景 | ゲーム全体の基本背景 | ダークブラウン(40, 30, 20) |
| power_map_background.png | 勢力図の背景 | 勢力マップ表示画面 | ダークグリーン(30, 40, 30) |
| battle_vs_background.png | 戦闘開始画面の背景 | VS画面（戦闘プレビュー） | 非常に暗い(20, 15, 10) |
| battle_combat_background.png | 戦闘中画面の背景 | 戦闘アニメーション画面 | 暗いブラウン(25, 20, 15) |
| battle_result_background.png | 戦闘結果画面の背景 | 戦闘終了後の結果表示 | ブラウン(30, 25, 20) |

## 画像の差し替え方法 / How to Replace Images

1. 実際の背景画像を用意します（1280x720px、16:9のアスペクト比）
2. 上記のファイル名に合わせて保存します
3. このフォルダ内の該当ファイルを上書き保存します
4. **重要**: ファイル名とサイズ（1280x720px）を必ず維持してください

## デザインガイドライン / Design Guidelines

### 推奨される背景デザイン

1. **main_background.png**
   - 戦国時代の城、城下町、または和風の風景
   - UI要素が重なるため、中央部分は比較的シンプルに
   - 上部と左右にはサイドパネル用のスペースを考慮

2. **power_map_background.png**
   - 古地図風のテクスチャ
   - 領地のポリゴンが重なるため、控えめな色調
   - 和紙や古紙のような質感が推奨

3. **battle_vs_background.png**
   - 緊張感のある暗い背景
   - 戦場、夜空、または炎の演出
   - 中央にVSテキストが表示されるため、中央は暗めに

4. **battle_combat_background.png**
   - 戦闘シーンに適した動的な背景
   - 戦場、草原、または城壁
   - テキストとバーが重なるため、コントラストに注意

5. **battle_result_background.png**
   - 結果を強調する背景
   - 勝利：明るめ、敗北：暗め の使い分けも可能
   - 大きなテキストが中央に表示されるため、背景は控えめに

### グリッド線について

プレースホルダー画像にはデザイン用のグリッド線（縦3分割、横2分割）が描画されています。
実際の背景画像ではこれらのグリッド線を参考に、UI要素の配置を考慮してデザインしてください。

## 注意事項 / Notes

- 現在のファイルはすべてプレースホルダー画像です
- 実際の背景画像に差し替える際は、PNG形式を推奨します
- 画像は透明部分を含まない完全な背景として扱われます
- 各画面の雰囲気に合わせて、異なる色調・テーマの背景を使用できます
- ファイルサイズを抑えるため、適度に圧縮することを推奨します（ただし画質に注意）

## 生成スクリプト / Generation Script

プレースホルダー画像の再生成が必要な場合は、プロジェクトルートで以下を実行：

```bash
python generate_placeholder_portraits.py
```

このスクリプトは肖像画と背景画像の両方を生成します。
