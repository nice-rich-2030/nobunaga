# 大名肖像画 / Daimyo Portraits

このフォルダには、ゲーム内で表示される大名の肖像画が格納されています。

## 画像仕様 / Image Specifications

- **サイズ**: 256 x 256 ピクセル (PNG形式)
- **ファイル名規則**: `daimyo_XX.png` (XXは大名IDの2桁表記)
- **形式**: PNG (透明背景対応可能)
- **背景色**: 金系(220, 200, 160) ※武将(青系)と区別するため

## 画像の差し替え方法 / How to Replace Images

1. 実際の大名肖像画を用意します（256x256px推奨）
2. 上記のファイル名規則に従って、対応する大名のファイル名に変更します
3. このフォルダ内の該当ファイルを上書き保存します
4. **重要**: ファイル名とサイズ（256x256px）を必ず維持してください

## 注意事項 / Notes

- 現在のファイルはすべてプレースホルダー画像です
- 実際の肖像画に差し替える際は、PNG形式を推奨します
- 大名用は金系背景(220, 200, 160)、武将用は青系背景(200, 200, 220)で区別されています
- 透明背景を使用する場合は、背景色と調和するようにしてください
- 新しい大名を追加する場合は、`daimyo_XX.png`の形式でファイルを追加してください

## 生成スクリプト / Generation Script

プレースホルダー画像の再生成が必要な場合は、プロジェクトルートで以下を実行：

```bash
python generate_placeholder_portraits.py
```

このスクリプトは`data/daimyo.json`と`data/generals.json`から人物データを読み込み、自動的にプレースホルダー画像を生成します。
