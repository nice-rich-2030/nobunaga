# 武将肖像画 / General Portraits

このフォルダには、戦闘演出画面で表示される武将の肖像画が格納されています。

## 画像仕様 / Image Specifications

- **サイズ**: 256 x 256 ピクセル (PNG形式)
- **ファイル名規則**: `general_XX.png` (XXは武将IDの2桁表記)
- **形式**: PNG (透明背景対応可能)
- **背景色**: 青系(200, 200, 220) ※大名(金系)と区別するため

## 画像の差し替え方法 / How to Replace Images

1. 実際の武将肖像画を用意します（256x256px推奨）
2. 上記のファイル名規則に従って、対応する武将のファイル名に変更します
3. このフォルダ内の該当ファイルを上書き保存します
4. **重要**: ファイル名とサイズ（256x256px）を必ず維持してください

## 注意事項 / Notes

- 現在のファイルはすべてプレースホルダー画像です
- 実際の肖像画に差し替える際は、PNG形式を推奨します
- 武将用は青系背景(200, 200, 220)、大名用は金系背景(220, 200, 160)で区別されています
- 透明背景を使用する場合は、背景色と調和するようにしてください
- 新しい武将を追加する場合は、`general_XX.png`の形式でファイルを追加してください

## 生成スクリプト / Generation Script

プレースホルダー画像の再生成が必要な場合は、プロジェクトルートで以下を実行：

```bash
python generate_placeholder_portraits.py
```

このスクリプトは`data/generals.json`と`data/daimyo.json`から人物データを読み込み、自動的にプレースホルダー画像を生成します。
