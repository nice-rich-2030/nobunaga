# 信長の野望風戦略ゲーム (Nobunaga's Ambition-style Strategy Game)

Pythonとpygameで作成された、戦国時代をテーマにした戦略シミュレーションゲームです。

![Game Version](https://img.shields.io/badge/version-1.0.0-blue)
![Python Version](https://img.shields.io/badge/python-3.8+-green)
![License](https://img.shields.io/badge/license-MIT-lightgrey)

## 📖 概要

このゲームは、戦国時代の大名となって領地を統治し、他の大名と戦いながら天下統一を目指す戦略シミュレーションゲームです。内政、軍事、外交、武将管理など、多様な要素を含んでいます。

## ✨ 主な機能

### 🏯 基本システム
- **領地管理**: 15の領地を管理（地形、城、石高、兵力など）
- **大名システム**: 複数の大名家が競い合う
- **武将システム**: 武力・統率・政治・知力を持つ武将を登用・配置
- **外交システム**: 同盟、不可侵条約、宣戦布告　 ・・・未実装

### ⚔️ 軍事システム
- **戦闘**: リアルタイム戦闘アニメーション
- **地形効果**: 山岳・森林・平地・沿岸で防御力が変化
- **城防御**: 城の有無で防御ボーナス
- **武将効果**: 武将の能力が戦闘力に影響

### 🌾 内政システム
- **開墾**: 開発レベルを上げて石高増加
- **町開発**: 町レベルを上げて商業収入増加
- **治水**: 洪水リスクを軽減
- **米配布**: 農民忠誠度を維持
- **徴兵**: 農民から兵士を徴兵

### 🔄 転送システム
- **資源転送**: 金・米・兵士を他領地へ即時転送
- **戦略的配置**: 後方から国境へ資源を集中

### 🤖 AIシステム
- **重み付きランダム判断**: 状況に応じて内政・軍事・転送を選択
- **地形考慮**: 防御ボーナスを考慮した攻撃判定
- **資源管理**: 後方領地から国境領地へ自動転送
- **武将配置**: 城や国境に優先的に武将を配置

### 🎭 イベントシステム
- **ランダムイベント**: 浪人訪問、豊作、洪水、一揆など
- **選択肢**: イベントの結果をプレイヤーが選択
- **イベント履歴**: 過去のイベントを確認可能

## 🎮 操作方法

### メイン画面
- **マウスクリック**: 領地選択、コマンド選択
- **Hキー**: イベント履歴表示

### コマンド一覧
1. **開墾** - 開発レベル向上（費用: 300金）
2. **町開発** - 町レベル向上（費用: 400金）
3. **治水** - 治水レベル向上（費用: 200金）
4. **米配布** - 忠誠度向上（費用: 100米）
5. **徴兵** - 兵士募集（費用: 1金/人）
6. **出陣** - 隣接領地へ攻撃
7. **転送** - 資源を他領地へ転送
8. **武将配置** - 武将の配置・解除
9. **外交** - 同盟・不可侵条約 ・・・未実装

### 戦闘画面
- **SPACE/クリック**: 戦闘スキップ
- **戦闘プレビュー**: 攻撃経路アニメーション（80フレーム）
- **戦闘アニメーション**: ラウンドごとの戦闘経過

## 🚀 インストール

### 必要要件
- Python 3.8以上
- pygame

### インストール手順

1. **リポジトリのクローン**
   ```bash
   git clone https://github.com/<your-username>/Nobunagas-ambition.git
   cd Nobunagas-ambition
   ```

2. **依存パッケージのインストール**
   ```bash
   pip install -r requirements.txt
   ```

3. **ゲーム起動**
   ```bash
   python main.py
   ```

## 🎯 ゲームの流れ

1. **織田家でゲーム開始**（尾張・美濃を支配）
2. **領地を選択**して内政・軍事コマンドを実行
3. **武将を登用**してイベントで配置
4. **隣接敵領地に出陣**して領土拡大
5. **外交で同盟**を結び戦略的優位を確保　 ・・・未実装
6. **ターン終了**でAI大名が行動
7. **経済収入**と**イベント発生**
8. **天下統一**を目指す

## 🔧 カスタマイズ

### 領地データの編集
`data/provinces.json`で領地の設定を変更可能：
- 領地名、位置、地形
- 初期石高、兵力、金
- 隣接領地の設定

### 武将データの編集
`data/generals.json`で武将の能力を調整：
- 武力、統率、政治、知力（0-100）
- 所属大名

### イベントの追加
`data/events.json`で新しいイベントを追加：
- イベントテキスト
- 発生条件
- 選択肢と効果

### ゲームバランス調整
`config.py`で各種パラメータを調整：
- コマンド費用
- 戦闘バランス
- 経済システム
- AI行動パラメータ
